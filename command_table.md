# VIM Command Cheet Sheet 

## 이동 
|명령어 | 기능 |
|------|------|
| j k h l     | 차례로 아래, 위, 왼쪽, 오른쪽 이동키 |
| b w | 단어 단위로 건너뛰는 명령, b는 문서 앞으로, w는 문서뒤로 |
| $ | 문자의 맨끝으로 이동하는 명령 |

## 편집
|명령어 | 기능 |
|------|------|
| .    | 이전에 했던 명령을 다시 반복함 |
| >G    | 현재 행부터 문서끝까지 들여쓰기를 함|


## 특수 기능
|명령어 | 기능 |
|------|------|
| :h <명령엉>    | 해당 명령어의 도움말을 보여줌 |
