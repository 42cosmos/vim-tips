## Tips
### 일반
- 처음 명령어가 익숙하지 않을 때는 `명령어 입력 모드`에서 `:h`를 통해 `help`를 잘 볼 수 있다.
  - 만약 특정 명령어에 대해 찾고 싶다면 `:h <찾고싶은 명령어>`를 입력하면 된다. 예를 들면 `:h G` 라는 식으로 검색이 된다.
